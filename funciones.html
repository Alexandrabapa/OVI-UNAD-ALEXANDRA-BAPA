HTML





¿Que es?

HTML cuyas siglas en ingles significa Hyper Text Markup Language, es un lenguaje basado en etiquetas que nos permite dar edición semántica a nuestros documentos de páginas web, cada vez que se desarrolla para web, nos encontramos con la necesidad de dar formato al texto a través de HTML, este lenguaje está estandarizado por la W3C, que es la organización que mane ja el lenguaje de la web. Recordemos entonces que HTLM no es un lenguaje de programación si no un lenguaje para modelar documentos, es decir le da forma a la información para poder ser presentada a las personas.

Por otro lado HTML5 es la versión 5 del lenguaje HTML, las caracterís ticas que han hecho especial esta edición de HTML, están relacionadas con englobar todas las tecnologías para las web en una sola, es decir con HTML5 no necesitamos tecnologías externas como Flash.

HTML se basa en etiquetas es así como interactúa con el navegador, el navegador interpreta cada etiqueta y las interpreta para mostrar de esta forma una información de una forma más amigable, estas etiquetas se identifican por que inician y terminan con los signos mayor y menor que < >, la estructura básica de HTML tiene una raíz que se llama html, luego se divide en dos segmentos Head la cual es la cabecera y contiene las etiquetas especiales que tienen que ver con la página que estas construyendo, pero que no se ven dentro del navegador como son el título, la descripción, la conexión con css o la conexión con java script. El siguiente segmento es Body, en donde se pone la información que se ve en el navegador, se utilizan etiquetas como <"h1"> para los títulos <"p"> para los parrafos, <"img src"> para las imágenes.

CSS es un lenguaje de hojas de estilo, no es un lenguaje de programación, nos permite controlar el aspecto y la presentación de cada elemento de una página web, nos permite modificar las propiedades Con CSS le damos forma color tamaño a nuestra página web
